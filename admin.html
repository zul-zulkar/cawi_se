ℹ️ Cara Kerja & Sinkronisasi
Penyimpanan per perangkat Password yang diubah disimpan di localStorage perangkat ini saja. Perangkat lain (HP/laptop petugas lain) tetap menggunakan password lama sampai admin mengubahnya di perangkat masing-masing.

Cara distribusi password baru:
1. Ubah password di halaman ini (di perangkat admin).
2. Sampaikan password baru kepada semua petugas.
3. Masing-masing petugas masuk ke …/admin.html dan atur password yang sama.
4. Atau: minta petugas logout (tutup tab/browser), lalu masukkan password baru di form login.
